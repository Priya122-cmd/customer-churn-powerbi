# Customer Churn Analysis — Power BI

## Project Overview
Analyzed 7,000 telecom customers to identify churn patterns across 
demographics, contract types, and billing methods.

## Dashboard Pages

### Page 1 — Customer Demographics Analysis
**KPIs:**
- Total Customers: 7K
- Churn Rate: 27%
- Avg Monthly Charges: $64.76
- Avg Tenure: 32.37 months

**Charts:**
- Customer Gender Distribution (donut chart)
- Churned vs Non-Churned Senior Citizens (bar chart)
- Customer Distribution by Tenure Group (bar chart)
- Churn Rate % by Tenure Segment (bar chart)
- Avg Monthly Charges by Tenure Group (line chart)
- Avg Monthly Charges by Tenure and Gender (line chart)

### Page 2 — Contract & Billing Insights
**KPIs:**
- Avg Monthly Charges: $64.76
- Avg Total Charges: $2.28K
- Churn Rate: 27%
- Churned Customers: 2K

**Charts:**
- Churn Rate % by Contract Type (month-to-month: 43%, two-year: 3%)
- Avg Total Charges by Contract Type
- Orders by Payment Method and Churn
- Avg Monthly Charges by Payment Method
- Paperless Billing vs Churn breakdown

**Filters:** Contract type slicer, Payment method slicer

## Tools Used
Power BI — DAX Measures, Power Query, KPI Cards, 
Multi-page Dashboard, Slicers, Data Modeling
